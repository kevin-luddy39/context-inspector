**Your AI workflow is failing 3 steps before you notice.**

↳ Live graphical bell curves (σ, mean drift, Gaussian fit)
↳ Domain-alignment scores as a leading indicator of degradation
↳ Per-chunk breakdowns so you can see exactly where contamination enters
↳ Attaches in seconds to any MCP server — Claude Desktop, Cursor, custom agents
